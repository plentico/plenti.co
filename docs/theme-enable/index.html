Theme Enable

Description

Enables an already downloaded theme that lives in your "themes" folder by simply making a "theme" entry in plenti.json. You can do this entirely manually if you'd like: copy a Plenti site from any source into your "themes" folder (for example themes/my-theme) then in plenti.json add "theme": "my-theme". 

Usage

Enable a theme with plenti theme enable THEME_FOLDER_NAME.
