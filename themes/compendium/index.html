Compendium

Repo: https://github.com/plenti-themes/compendium
Built by: roobyz
Designed by: roobyz
Start a new project using this theme:
plenti new site my-site --theme=https://github.com/plenti-themes/compendium
Or add this theme to an existing project:
  1. plenti theme add https://github.com/plenti-themes/compendium
  2. plenti theme enable compendium
  3. Make sure you manually copy any route overrides and delete conflicting content or layouts.
