Fitlife

Repo: https://github.com/plenti-themes/fitlife
Built by: jamestagal
Designed by: codewithsadee
Start a new project using this theme:
plenti new site my-site --theme=https://github.com/plenti-themes/fitlife
Or add this theme to an existing project:
  1. plenti theme add https://github.com/plenti-themes/fitlife
  2. plenti theme enable fitlife
  3. Make sure you manually copy any route overrides and delete conflicting content or layouts.
