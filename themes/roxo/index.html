Roxo

Repo: https://github.com/plenti-themes/roxo
Built by: Jantcu
Designed by: StaticMania
Start a new project using this theme:
plenti new site my-site --theme=git@github.com:plenti-themes/roxo
Or add this theme to an existing project:
  1. plenti theme add git@github.com:plenti-themes/roxo
  2. plenti theme enable roxo
  3. Make sure you manually copy any route overrides and delete conflicting content or layouts.
